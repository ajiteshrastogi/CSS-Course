Ans3: In CSS Flexbox, align-self has a higher priority than align-items for individual flex items. Beacause
align-items: This property applies to the flex container and sets the default alignment for all flex items inside that container. It acts as a "group rule" for alignment.
align-self: This property applies directly to a specific flex item and overrides the alignment set by align-items for that particular item.
